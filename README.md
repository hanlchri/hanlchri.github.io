# Hanley's Hood Reimagined

> A modern reimagining of the classic classroom game.

## Overview

A web-based remake of Mr. Hanley's Hood, updated with modern web technologies, smooth animations, and a polished UI. This project preserves the nostalgia of the original while bringing it into the modern era.

## Tech Stack

React, TypeScript, Vite, Framer Motion, Shadcn UI, React Query, Recharts, Tailwind CSS
